### 💻 Host: Ubuntu 24.04 (WSL2 or Native)  
### 🐳 Docker Image: `osrf/ros:melodic-desktop-full`  
### 🎯 Target Package: [omorobot/omo_r1](https://github.com/omorobot/omo_r1)  
  
### Issue Tracking  
- If problems arise while following the process below, it is recommended to refer to the official documents in Method 1  
	- However, for official documents, you may need to change the code because you are installing ROS Melodic directly in Ubuntu 18.04  
  
---  
  
## 1. System Preparation (Host)  
  
### Confirm Docker Installation  
```bash  
docker --version  
```  
  
### Verifying WSL2 Settings  
```bash  
wsl -l -v  
# Verify Ubuntu is version 2  
```  
  
### Create Catkin Workplace Directory (Host)  
```bash  
mkdir -p ~/r1/catkin_ws/src  
```  
  
### X11 GUI Forwarding Setting (Host)  
```bash  
sudo apt install -y x11-xserver-utils  
xhost +local:docker  
```  
  
> ⚠️ Reboot each time and run again  
  
---  
  
## 2. Docker configuration settings  
  
### Download ROS Docker Image  
```bash  
docker pull osrf/ros:melodic-desktop-full  
```  
  
### Running a container  
```bash  
docker run \  
--name ros1 \  
-it \  
--privileged \  
--env="DISPLAY=$DISPLAY" \  
-v=/tmp/.X11-unix:/tmp/.X11-unix:ro \  
-v=/dev:/dev \  
-v=~/r1/catkin_ws:/root/catkin_ws \  
-w=/root/catkin_ws \  
osrf/ros:melodic-desktop-full  
```  
  
---  
  
## 3. Container Internal Settings  
  
### Installation of required packages  
```bash  
apt-get update && apt-get install -y \  
python \  
python-pip \  
python-setuptools \  
python-catkin-tools \  
python-rosdep \  
python-rosinstall \  
python-rosinstall-generator \  
python-wstool \  
build-essential \  
git  
```  
  
---  
  
## 4. Initialize and update rosdep  
  
```bash  
rosdep init || echo "already initialized"  
rosdep update  
```  
  
---  
  
## 5. Initialize Catkin workspace  
  
```bash  
cd /root/catkin_ws  
catkin_make  
```  
  
---  
  
## 6. Register ROS Environmental Variables  
  
```bash  
echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc  
echo "source /root/catkin_ws/devel/setup.bash" >> ~/.bashrc  
source ~/.bashrc  
```  
  
---  
  
## 7. Package Clones  
  
```bash  
cd ~/catkin_ws/src  
git clone https://github.com/omorobot/omo_r1.git  
git clone https://github.com/PinkWink/ydlidar.git  
```  
  
---  
  
## 8. Dependency Installation  
  
```bash  
cd ~/catkin_ws  
rosdep install --from-paths src --ignore-src -r -y  
```  
  
> Try again in case of rosdep error or check the network  
  
---

## 10. Final build  
  
```bash  
cd ~/catkin_ws  
catkin_make  
```  
  
---  
  
## 11. Run roscore  
  
```bash  
roscore  
```  
  
- End with 'Ctrl + C'  
- Background Run: 'score &'  
  
---  
  
## ## 12. Connecting containers from additional terminals  
  
```bash  
docker exec -it ros1 /bin/bash  
```  
  
> subsequent rosrun, roslaunch commands available
